1.Installing python

There are many ways you can install Python and the libraries required for this class, depending on your system and experience with command-line tools. We will be using Python 2.7! There is python 3.x, but many packages are not supported in that version, and others are not supported in older versions of python.

The easiest is to use the Enthought distribution (Website), which has nearly all of the packages required for the class

Another is to install python (Website), and then install pip (a package installer; Mac: Website; Windows: Website£©

From there you need to be sure you have all of the packages required for the class. To test a package, simply start python (you¡¯ll see this prompt: >>>) and type import [package name]. If it does not give you an error, you have the package installed.
